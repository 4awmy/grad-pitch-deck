Slide 2: What We Build
Edge-AI Provisioner: Automated Constraint-Aware Engine

The Core System: A self-contained local application that automates and optimizes the design and deployment of local AI environments based on hardware constraints.

1. Automated Hardware Auditing
Reads host specs and available VRAM/RAM without manual user input.
2. Constraint-Aware Routing
Matches business needs to hardware-appropriate open-weight models using constraint satisfaction.
3. Actionable Execution
Delivers a ready-to-deploy Docker Compose package or installation script tailored to the machine.
