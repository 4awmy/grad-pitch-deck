Hardware · Accessibility

Sign Language Glove

Bridging the communication gap for Egypt's deaf community — in real time.

Arduino / ESP32 Flex Sensors MPU6050 IMU TensorFlow Lite React Native BLE

The Problem

The deaf and hard-of-hearing community in Egypt face near-total communication barriers in standard daily interactions. Face-to-face services (like banking, clinical care, and general government interactions) lack sign language support, and current software solutions require continuous typing or screen interaction.

What We Build

A wearable glove integrating flex sensors and an inertial measurement unit (IMU) to map fingers and wrist motion. Gesture patterns are classified locally on-device or streamed via BLE to a companion smartphone app which instantly outputs localized spoken Arabic.

How It Works

01

Data Acquisition

Flex sensors measure knuckle bending while an MPU6050 accelerometer/gyroscope captures spatial hand rotation and velocity vectors.

02

Signal Processing

An on-board microcontroller (ESP32/Arduino) filters mechanical noise, normalizes raw sensor inputs, and formats gesture packages.

03

ML Classification

A lightweight TensorFlow Lite model evaluates the sensor time-series data to classify sign gestures from the Egyptian Sign Language vocabulary.

04

Speech Generation

The companion React Native mobile app receives the gesture codes over BLE and synthesizes clear, spoken Arabic output instantly.

Technology Mapping

ESP32
Provides low-energy operation, sensor interfacing pins, and built-in BLE support.
TensorFlow Lite
Enables edge ML execution on smartphones for sub-100ms gesture classification.
React Native
Coordinates the speech synthesis service and displays the history interface.

Social Impact Areas

Dataset Growth
Contributing a clean, open-source dataset of Egyptian sign gestures to help local ML researchers.
Hospital Integration
Direct deployment in patient intake areas to assist deaf patients communicating symptoms.
Educational Access
Allowing deaf children to practice and learn sign gestures through interactive games.

Why This is Publishable

We present a novel spatial sensor fusion model optimized for Arabic Sign Language. We validate gesture classification accuracy under physical sensor placement variances and variable motion velocities.

Target Publication Venues

Targeting IEEE EMBC (Engineering in Medicine and Biology Conference) or ACM Pattern Recognition Letters. Emphasizing low-cost wearable assistive technology.
