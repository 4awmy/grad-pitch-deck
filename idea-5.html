StudyX

Your lecture notes, transformed into an adaptive exam trainer.

React Supabase OpenAI API LangChain Expo

The Problem

Students have access to more video lectures, slides, and textbooks than ever, but lack structured methods to retain it. Passive reviewing (highlighting text, re-reading files) produces poor learning retention and exam results compared to active recall.

What We Build

An interactive, AI-driven study companion. Students upload PDFs of lecture slides or notes. StudyX extracts core entities, automatically builds spaced-repetition flashcards, generates adaptive practice quizzes, and hosts interactive exam simulations.

How It Works

01

Document Parsing

Students upload a lecture slide deck or textbook chapter to the web app dashboard.

02

Generative Structuring

An LLM parser extracts key academic definitions, historical dates, equations, or code concepts, formatting them into structured card decks.

03

Spaced Repetition

Our algorithm schedules card review intervals (like SuperMemo-2) to display card reviews just before memory decay thresholds occur.

04

Simulated Exams

StudyX generates timed Mock MCQ and short-answer exams, scoring responses and highlighting specific conceptual gaps.

Technology Mapping

Supabase
Handles secure student accounts, card histories, schedules, and real-time database synching.
OpenAI API
Processes text extractions to generate consistent question-and-answer templates.
Expo (Planned)
Provides the cross-platform compiler to target native mobile systems (Android/iOS) later on.

Why Not Just Use ChatGPT?

Source Grounded
Questions are strictly grounded in uploaded course files, avoiding hallucinations.
Adaptive Algorithms
Built-in spaced repetition card queues adapt dynamically to how easily you recall answers.

Why This is Publishable

We analyze the impact of LLM-generated flashcards vs. human-created decks on student retention. We measure quantitative grade improvement between control groups using traditional study methods and groups using StudyX.

Target Publication Venues

Targeting Elsevier Computers & Education or IEEE Transactions on Learning Technologies (TLT).
